# File-Manager-Indexe
Add tags/keywords/description  to the file  , and use the app to search for it using its tags or by description


```Java
String titre : indexeur de recherche


String problématique : 'je veux cherche un fichier mais je ne sais pas ou '

String utilité : 
    - uploader un fichier pour le sauvegarder dans l'application
    - ajouter des mots clé ou une description ou bien des 'tags'
    - si le fichier apparaît dans les résultats de recherche , clicked pour l'ouvrir dans l'explorateur
```
